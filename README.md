# NHS Interview Coding Challenge
This is the GitHub repository for my NHS interview coding challenge which started on the 5th May 2026. This solution uses the .NET 10 SDK so the runtime may need to be installed on your PC.

I started this solution within Visual Studio Code, so hopefully it should still look and compile fine in Visual Studio. The mocked data located at `NHSInterviewCodingChallenge/Data/MOCK_DATA.csv` was generated at Mockaroo.com.

Some unit tests for the controller and the service exist in NHSInterviewCodingChallenge.Tests.
